# LocalMeetingAI
全离线私有化AI会议助手，数据100%不出内网
- 实时收音转写，自动识别发言人
- 会议全程AI主动实时输出观点、风险、补充方案
- 项目隔离私有RAG知识库，上传项目文档联动分析
- 单机离线/企业内网双部署模式，无第三方云端上传
- 完全开源 MIT License，免费商用、二次开发

## 核心能力
1. 音频采集：系统会议音频/麦克风本地降噪处理
2. 离线ASR实时字幕，区分多参会人
3. 按项目独立知识库，文档完全隔离不互通
4. 流式大模型实时推理，无需手动触发AI发言
5. AES256本地文件加密，全程无外网数据传输
6. Windows/Mac跨端桌面客户端Tauri

## 两种部署方案
1. 单机个人版：本地GPU/CPU运行，断网可用，适合小组
2. 企业内网集群版：Docker/K8s部署，多账号权限、审计日志

## 硬件最低要求
### 单机版
最低：i7/R7 CPU + 16G内存（CPU低速推理）
推荐：32G内存 + RTX3060/4060 12G显存

### 企业服务端
24G显存GPU服务器、内网NAS存储
